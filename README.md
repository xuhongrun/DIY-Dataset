# DIY-Dataset

## Getting Started

### Windows
```shell
pip install -r requirement.txt
```

### Linux
```shell
wget http://chromedriver.storage.googleapis.com/89.0.4389.23/chromedriver_linux64.zip
unzip chromedriver_linux64.zip

pip3 install -r requirement.txt
```
